# PawNextDoor (Prototype)

A portable Vite + React + Tailwind mockup for the PawNextDoor concept.

## Quick Start
```bash
npm install
npm run dev
```

Open the printed localhost URL in your browser. Switch screens with the top navigation.

## Notes
- Payment is mocked (no backend).
- Photo uploads are client-side previews only.
- Designed for rapid iteration and investor demos.
